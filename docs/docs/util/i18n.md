!!! tip "xtyle.i18n"

    Register globally an Object with translations with **`xtyle.translations`** to later use those translations inside your project with **`xtyle.i18n`**


```js
/* Set Translations */
xtyle.translations({
  fr: {
    greetings: {
      hello: "Bonjour!",
    },
  },
});


/* Preview */
console.log("i18n: ", xtyle.i18n("fr.greetings.hello"));
```
